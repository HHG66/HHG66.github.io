一、学习总结

简介：

网页：

1、 什么是网页：网站时网页的集合，通常是HTML格式的文件。它要通过游览器阅读。网页是构成网站的基本元素，它通常由图片、文字、视频、声音等等信息组成。

2、 什么是HTML：描述网页的语言，不是编程语言是一种超文本标记语言，用于标记元素

游览器

1、 常用游览器：Chrome、IE、火狐、edge游览器、safari游览器等等、

2、 游览器内核：也称为渲染引擎，负责读取网页，整理信息，计算网页的显示方式并显示网页。

Chrome：blink内核

Web标准

1、Web标准：由W3C组织制定的一系列标准的集合。用来规定页面排版和显示。

2、web标准构成：结构、表现、行为

​ 结构：元素的整理和分类

​ 表现：元素的排版、颜色、大小等外观样式，主要是指CSS

​ 行为：网页模型和交互，主要是指Javascipt

要求：结构、表现、行为相分离

相对路径：从所编辑的html文件到目标文件的路径

表单标签

表单：收集用户信息

Html中表单由三部分组成分别是：表单域、表单控件、提示信息三部分

action URL 规定当提交表单时向何处发送表单数据。
method get post 规定用于发送 form-data 的 HTTP 方法。
name form_name 规定表单的名称。
enctype application/x-www-form-urlencoded
multipart/form-data
text/plain		 enctype 属性规定在发送到服务器之前应该如何对表单数据进行编码

<input> 单标签，标签用于搜集用户信息。

属性：

type button（后期搭配js使用） checkbox（复选框） file（上传文件） hidden image password（不可见的密码） radio（单选） reset（重置） submit（提交按钮） text（文本） 规定 input 元素的类型。
name field_name 定义 input 元素的名称。
value Value（得到表单数据发送到服务器） 规定 input 元素的值。
checked checked 规定此 input 元素首次加载时应当被选中。
maxlength number 规定输入字段中的字符的最大长度。

Name：同一个标签下值必须相等，属性可以为多选一提供单独的值，使得不能全部都选！只要给后台开发人员使用

Value：给后台人员使用

Lable标签

label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。就是说，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上。

for id 规定 label 绑定到哪个表单元素。

Select下拉表单

select 元素可创建单选或多选菜单。

option 元素定义下拉列表中的一个选项（一个条目）。

浏览器将 <option> 标签中的内容作为 <select> 标签的菜单或是滚动列表中的一个元素显示。

option 元素位于 select 元素内部。是成对属性

selected selected 规定选项（在首次显示在列表中时）表现为选中状态。
